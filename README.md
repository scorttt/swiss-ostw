# Swiss
 An Overwatch workshop mode built on [OSTW](https://github.com/ItsDeltin/Overwatch-Script-To-Workshop) to provide a slew of quality-of-life features for Overwatch content creators

## Features
- Player recording with dummy bot playback
- Timeline and Editor
- Noclip
- Timescale adjustment
- Aim assist
- Camera pathing with B-spline curves
- Cursor navigation for all menus
